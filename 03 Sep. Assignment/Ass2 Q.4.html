1. html : The root element of an HTML document.
    It contains all other HTML elements on the page.

  2. head : Contains meta-information about the HTML document.
    Includes elements like title, meta, and link.

  3. title : Sets the title of the web page, which appears in the browser's title bar or tab.

  4. meta : Provides metadata about the HTML document, such as character encoding and keywords.

  5. link : Used to link external resources like stylesheets (CSS) to the HTML document.

  6. style : Contains inline CSS code for styling HTML elements within the document.

  7. script : Used to embed or reference JavaScript code within the HTML document.

  8. body : Contains the visible content of the web page, including text, images, and other elements.

  9. h1, h2, h3, h4, h5, h6 : Heading tags, where h1 represents the highest level of heading and h6 the lowest.
    Used to structure content and define headings on the page.

  10. p : Defines a paragraph of text.
    Commonly used for grouping and styling text content.

    11. a : Creates hyperlinks to other web pages or resources.
    Requires an href attribute to specify the destination URL.
  11. img : Embeds images within the web page.
    Requires a src attribute to specify the image file source.

  12. ul : Creates an unordered (bulleted) list.
    Contains li elements for each list item.

  13. ol : Creates an ordered (numbered) list.
    Also contains li elements for each list item.
